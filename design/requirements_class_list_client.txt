eZ Find: Class list client	
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
:Author:   Kåre Køhler Høvik
:Revision: 
:Date:    


Introduction
============

Description
-----------
In order for eZ Find to be able to search external installations, it'll need to
know the class list and class attribute list on the extenral installation.
The class list client contains functionality for reading the class list and
class attribute list from the external installation.


Requirements
============
- ReST client for reading class list from Class list service
- Caching of remote class list


Considerations
==============


Requirements
============
ReST Web service for getting class list.
